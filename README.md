This project predicts the market price of houses based on input features such as number of rooms, area, location, and other property characteristics.

Files in this repository:
	•	House_Price_Prediction.ipynb → Jupyter notebook containing data preprocessing, model training, and prediction steps.
	•	housing_data.csv → Dataset used for model training and testing.

Steps performed:
	1.	Data cleaning & preprocessing
	2.	Exploratory Data Analysis (EDA)
	3.	Feature engineering (handling categorical and numerical features)
	4.	Model training using XGBoost Regressor
	5.	Model performance evaluation using R² Score and RMSE

Tools Used:
	•	Python
	•	Pandas, NumPy
	•	Matplotlib 
	•	Scikit-learn
	•	XGBoost
